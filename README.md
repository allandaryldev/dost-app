# **DOST HANDS ON APPLICATION**

# LARAVEL 9+
# VUE 3+

## **REQUIREMENTS**
1. Node js version 14+ installed
2. Composer 2+ installed 
3. Apache Serve
    - xampp
    - mamp
    - laragon
4. Mysql
    - phpmyadmin

5. Code Editor

### RUNNING LOCALLY
1. clone the project
```bash
```
2. cd to the project
``` bash
    cd my-project
```
3. install dependencies
```bash
    composer install
```
```bash
    npm install
    or
    yarn install
```
4. run the project
```bash
    npm run dev 
    or
    yarn dev
```
